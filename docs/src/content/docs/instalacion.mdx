WN++ se distribuye como un binario único, sin dependencias. Lo instalas y funciona altiro.

## Instalación rápida

<Tabs>
  <TabItem label="macOS / Linux" icon="linux">

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/cuervolu/wn/releases/latest/download/wn-cli-installer.sh | sh
```

El script detecta tu arquitectura automáticamente — Intel, Apple Silicon, ARM — y descarga el binario correcto.

  </TabItem>
  <TabItem label="Windows" icon="windows">

Abre PowerShell y ejecuta:

```powershell
powershell -ExecutionPolicy Bypass -c "irm https://github.com/cuervolu/wn/releases/latest/download/wn-cli-installer.ps1 | iex"
```

  </TabItem>
</Tabs>

## Verificar la instalación

```bash
wn --version
```

Si ves algo como `wn 0.1.0`, estás listo.

## Instalación manual <Badge text="opcional" variant="note" />

Si prefieres no usar scripts porque no confías en mis habilidades como programador de bash, descarga el binario directamente desde [GitHub Releases](https://github.com/cuervolu/wn/releases/latest) y agrégalo a tu `PATH`.

| Plataforma            | Archivo                                          |
|-----------------------|--------------------------------------------------|
| macOS Apple Silicon   | `wn-cli-vX.Y.Z-aarch64-apple-darwin.tar.gz`       |
| macOS Intel           | `wn-cli-vX.Y.Z-x86_64-apple-darwin.tar.gz`        |
| Linux x86_64          | `wn-cli-vX.Y.Z-x86_64-unknown-linux-gnu.tar.gz`   |
| Linux ARM64           | `wn-cli-vX.Y.Z-aarch64-unknown-linux-gnu.tar.gz`  |
| Windows 64-bit        | `wn-cli-vX.Y.Z-x86_64-pc-windows-msvc.zip`        |

## Compilar desde el código fuente <Badge text="avanzado" variant="caution" />

Si quieres la versión más reciente del repositorio o simplemente tienes curiosidad de cómo está hecho (o me querí robar la wea)

<Steps>

1. **Instala Rust**

   Necesitas Rust 1.80 o superior. Si no lo tienes:

   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

2. **Clona el repositorio**

   ```bash
   git clone https://github.com/cuervolu/wn
   cd wn
   ```

3. **Compila en modo release**

   ```bash
   cargo build --release
   ```

4. **Ejecuta**

   ```bash
   ./target/release/wn
   ```

</Steps>

<Aside>
  El primer `cargo build` descarga dependencias y puede tomar un par de minutos.
  Los builds siguientes son considerablemente más rápidos.
</Aside>

## Actualizar

WN++ incluye un comando de actualización incorporado:

```bash
wn update
```

Descarga la última versión disponible y reemplaza el binario actual.

:::note[Migración desde Piola]
Si tenías instalado el binario antiguo `piola`, reinstala WN++ con el script nuevo. El comando `piola update` no migra al nuevo nombre ni mueve archivos a `~/.wn/bin`; instala `wn`, confirma que funciona y luego elimina la instalación antigua cuando ya no la necesites.
:::

## Primeros pasos

Con WN++ instalado, tienes dos formas de usarlo:

**REPL interactivo** — escribe expresiones y ve el resultado de inmediato:

```bash
wn
```

```
WN++ v0.1 — escribe 'chao' para salir
>>> 1 + 1
2
>>> altiro("Wena!")
Wena!
>>> chao
¡Chao!
```

**Ejecutar un archivo** — crea un archivo con extensión `.cl`:

```bash
wn mi_programa.cl
```

```wn
// mi_programa.cl
wea nombre = "Tulio"
altiro("Wena, " + nombre + "!")
```

```
Wena, Tulio!
```
